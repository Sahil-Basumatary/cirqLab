# CirqLab

> A platform for predicting extrachromosomal DNA (ecDNA) dynamics and therapy resistance in cancer.

## What is CirqLab?

CirqLab is a computational platform that predicts how extrachromosomal DNA (ecDNA) evolves under therapy pressure, helping researchers and clinicians understand and combat cancer drug resistance.

## Tech Stack

- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS
- **Backend:** FastAPI + Python 3.11+
- **ML Core:** PyTorch + PyTorch Geometric
- **Database:** PostgreSQL + Redis
